花样流水灯（汇编程序）
作者：佘晓凯

本作品是基于普中51开发板的花样流水灯
MCU：	STC89C52RC―90C
外设：	P1.0-P1.2	按键	*3
		P2			LED		*8	（共阴）
		74HC245		
		晶振		11.0592M
		采用6T模式
		

本程序共包含5个源文件及其对应头文件
*请在目标选项配置A51中设置包含文件路径

本程序功能说明：
	本程序为花样流水灯，共有3个按键（以下p1.0-p1.2将分别用k1-k3代表），
k1-k3作为模式选择按键，k1为闪烁灯模式，k2为流水灯模式，k3为呼吸灯灯模式，
在同模式下按按键（如闪烁灯模式下按k1），则为改变状态，闪烁灯为改变闪烁灯
位置，流水灯为改变流水的方向，呼吸灯为改变呼吸频率


用到的缓存
1.MODE	0x30	纪录当前模式
2.COUNT	0x31	模式1与模式2的频率计数，每500ms改变一次状态
3.TOCNT	0x32	纪录T0定时器中断进入的次数
4.MODE2LOR		BIT 	00H		模式2的流水灯方向
5.AOD			BIT		01H		模式3输出pwm占空比的增减
6.R3			当前闪烁的灯
7.R2			当前流水的位置
8.R4			当前中断次数，（20次一个周期）
9.R5			需要输出低电平的中断次数（控制占空比）
10.R6			当前占空比维持的周期数（需要维持R7个周期）
11.R7			当前占空比需要维持的周期数（控制呼吸频率）
